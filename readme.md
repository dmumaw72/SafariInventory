#Example Inventory Management Example

###Overview
This is a sample inventory management system for small companies (20 employees or less).  Employees will be able to log into the system to adjust inventory
levels, create orders, add customers, and view inventory reports
###Features
*User signup, log in, and authentication
*User roles
*Full CRUD capabilities
*View top customers by sales, and top selling products
###Technologies
*C#
*Razor Templates
*SQL Server
*BootStrap
*Javascript
*HTML
*CSS
